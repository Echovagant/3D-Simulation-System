# 3D场景编辑与仿真系统

欢迎使用3D场景编辑与仿真系统，本系统提供了一套完整的工具链，用于创建、编辑3D场景，并进行物理仿真和AI交互实验。

## 系统功能概述

本系统包含以下核心模块：

1. **场景编辑器**：用于构建和编辑3D场景，添加模型、灯光和物理属性
2. **仿真实验室**：运行和观察物理仿真效果，调整参数并分析结果
3. **模型库**：管理3D模型资产，支持导入、预览和分类管理
4. **AI演武场**：实现AI智能体与3D环境的交互，测试和训练AI行为

## 快速开始

### 第一步：创建场景

1. 点击导航栏中的"场景编辑器"进入场景编辑界面
2. 使用左侧工具栏添加基础几何体或从模型库导入模型
3. 调整模型位置、旋转和缩放，构建您的3D场景
4. 添加灯光和相机，设置场景环境

### 第二步：设置物理属性

1. 在场景编辑器中选择需要添加物理属性的物体
2. 在右侧属性面板中启用物理特性
3. 设置质量、摩擦系数和 restitution（弹性）等参数
4. 保存场景

### 第三步：运行仿真

1. 点击导航栏中的"仿真实验室"
2. 加载您创建的场景
3. 点击"开始"按钮运行物理仿真
4. 可以调整仿真速度、视角等参数观察效果

### 第四步：添加AI交互（可选）

1. 点击导航栏中的"AI演武场"
2. 加载场景并添加AI智能体
3. 配置AI行为规则或导入训练模型
4. 运行并观察AI与环境的交互效果
